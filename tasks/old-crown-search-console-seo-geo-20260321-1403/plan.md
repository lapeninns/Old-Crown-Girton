# Implementation Plan: Old Crown Search Console SEO + GEO

## Objective

We will use Search Console data to improve Old Crown's local discoverability, reduce page-level cognitive load, and strengthen easy conversion from organic traffic.

## Success Criteria

- [x] Search Console opportunities are translated into concrete on-site improvements.
- [x] Geo/local intent is better represented in metadata, content hierarchy, or internal linking.
- [x] Conversion routes become simpler and easier to scan.
- [x] Build verification passes.

## Testing Strategy

- Search Console export analysis
- Production build verification
- Manual route QA when browser tooling is available

## Rollout

- Analyze search data first, then implement the highest-leverage SEO and conversion improvements in one focused pass.
- Prioritize technical consolidation plus page-snippet improvements over large content rewrites.
